# 🚀 AI Sales Forecasting & Business Intelligence System

An end-to-end **AI-powered Sales Forecasting and Business Intelligence System** built using **Python, Streamlit, Power BI, SQL, Machine Learning, and Excel**. This project transforms raw sales data into actionable business insights through interactive dashboards, predictive analytics, and intelligent sales forecasting.

The system integrates **SQL for business analysis**, **Power BI for interactive dashboards**, and **Machine Learning (XGBoost & LSTM)** for demand prediction and future sales forecasting, enabling data-driven business decisions.

---

# 📌 Project Overview

Businesses generate massive amounts of sales data every day, but extracting meaningful insights from that data is challenging.

This project addresses that problem by combining **Business Intelligence** and **Artificial Intelligence** into one integrated solution that allows users to:

- Analyze historical sales performance
- Monitor business KPIs
- Predict future sales
- Forecast product demand
- Identify profitable products and regions
- Generate inventory recommendations
- Support strategic business decisions
